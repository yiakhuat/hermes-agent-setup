01 — OVERVIEW

Project Overview

This project extends a self-hosted Hermes Agent instance with two custom MCP (Model Context Protocol) tools that run as persistent stdio servers on the VPS.

📅
icloud-calendar
Read, create and update iCloud calendar events via Apple's CalDAV API.
🎯
job-scout
Search job boards, filter by saved preferences, and return a clean ranked digest.

Both tools are registered permanently in Hermes' config.yaml and survive restarts and new sessions. The Job Scout tool additionally stores user preferences in a local JSON file — set once, remembered forever.

04 — ICLOUD CALENDAR MCP

iCloud Calendar MCP Tool

Connects to Apple's CalDAV API and exposes calendar management capabilities to Hermes.

MCP Tools Exposed

Tool NameDescription
mcp_icloud_calendar_list_eventsList upcoming events by date range and calendar name
mcp_icloud_calendar_create_eventCreate a new event in any iCloud calendar
mcp_icloud_calendar_update_eventModify an existing event by UID

Configuration in config.yaml

yaml
mcp_servers:
  icloud-calendar:
    command: /root/hermes-venv/bin/python3
    args:
    - /root/icloud_calendar_mcp.py
    timeout: 60
    connect_timeout: 60
    env:
      ICLOUD_EMAIL: your@icloud.com
      ICLOUD_PASSWORD: your-app-specific-password
⚠

Use an App-Specific Password from appleid.apple.com — not your regular iCloud password. Go to Security → App-Specific Passwords.

05 — JOB SCOUT MCP

Job Scout MCP Tool

Searches multiple job boards, filters against saved preferences, deduplicates across sources, and returns a clean ranked digest. Preferences persist to disk — set once, remembered forever.

Architecture

User asks Hermes → Hermes calls MCP tool → job_scout_mcp.py ↓ Load job_profile.json ↓ ┌───────────────────────┴────────────────────────┐ ↓ ↓ Adzuna API JSearch API (250 req/mo free) (200 req/mo free) ↓ ↓ └───────────────────────┬────────────────────────┘ ↓ Filter → Rank → Deduplicate ↓ Save to seen_jobs.json ↓ Return clean digest to Hermes

Persistent Storage Files

FilePurpose
job_profile.jsonSaved user preferences — loaded on every search
seen_jobs.jsonDeduplication cache — prevents showing same job twice

06 — REGISTERING TOOLS

Registering MCP Tools with Hermes

Step 1 — Edit config.yaml

bash
nano ~/.hermes/config.yaml

Append both MCP server entries at the bottom under the mcp_servers: key.

Step 2 — Restart Hermes Gateway

bash
systemctl --user restart hermes-gateway
⚠

Always restart the gateway after any config.yaml change for tools to reload.

Step 3 — Verify Registration

bash
hermes tools list

Expected output under MCP servers:

MCP servers: icloud-calendar all tools enabled job-scout all tools enabled
07 — PERSONALITIES

Hermes Personalities

Personalities allow Hermes to switch roles and behave differently depending on the task. Activated with /personality <name> inside chat.

yaml
personalities:
  job-scout:
    prompt: "You are a job search specialist. Always use the job-scout MCP tool."
  calendar-assistant:
    prompt: "You are a calendar assistant. Always use icloud-calendar tools."
  tech-writer:
    prompt: "You are a professional technical writer. Write clear Markdown documentation."
  doc-reviewer:
    prompt: "You are a strict technical documentation reviewer. Check clarity and accuracy."

Switching Personalities

prompt
/personality job-scout
/personality calendar-assistant
/personality tech-writer
